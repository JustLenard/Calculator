# Calculator

This is not a simple calculator.  
  
No, my version of it can do complex logic. It fully supports parenthesis and power.  
 
Inputs like this:  
    
2 * (8 ^ (3 - 2 * 0.5 + 9 ^ 2 / 3) + 5) ^ 0.5 - 5  
  
are fully supported.  
Live Demo:    
https://justlenard.github.io/Calculator/

P.S.: pss, the answer is 1.
